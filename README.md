# Generate-Script-SQL 💾

**Generate-Script-SQL** es una herramienta de escritorio potente e intuitiva desarrollada en **Python** con **Tkinter**. Su objetivo es automatizar la creación de scripts SQL para bases de datos MySQL, permitiendo a los desarrolladores generar sentencias complejas de forma visual y rápida desde una interfaz gráfica moderna.

![Interfaz del Proyecto](sql-1.jpeg)

## 🚀 Características principales

* **Gestión de Estructuras (DDL):** Genera scripts para `CREATE DATABASE` y `CREATE TABLE` con soporte para motores (InnoDB/MyISAM), Charsets y Collation.
* **Manipulación de Datos (DML):** Generador visual para sentencias `INSERT`, `UPDATE`, `DELETE` y `SELECT`.
* **Objetos Avanzados:** Módulos dedicados para la creación de `PROCEDURES`, `VIEWS` y `TRIGGERS`.
* **Configuración de Campos:** Permite definir tipos de datos, restricciones (Primary Key, Unique, Auto Increment) y nulidad con un par de clics.
* **Previsualización en tiempo real:** El script se genera instantáneamente en un área de texto lista para copiar y usar en cualquier gestor de base de datos.

## 🛠️ Tecnologías utilizadas

* **Lenguaje:** Python 3.x
* **Interfaz Gráfica:** Tkinter / CustomTkinter
* **Base de Datos:** MySQL (Sintaxis compatible)
* **Librerías adicionales:** `tkinter` para el desarrollo de la GUI.
